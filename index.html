Initializing Git

Install and create the environment for using Git on one's computer
  1. Download Git
  2. Click next through all prompts (except the path to enable Git to run in Windows CMD as well)
  3. Launch Git
  4. Add one's email address to Git
    • Type: "git config --global user.email 'email_address@example.com'"
  5. Add one's user name to Git
    • Type: "git config --global user.name 'One's username'"

Initializing Git repositories

Create repositories on one's local system and adding Git to the project folder
  1. Create the repository for the project
  2. Open that folder in Git Bash
    • Either type: "cd directory1/directory2/myApplication" or just right-click on the folder and click "Git Bash Here"
  3. Create one's .git file and initialize git in the folder
    • Type: "git init"
  4. Create the repository on Github
    • Copy the link of the remote repository as well, we will need that for the next step
  5. Add the remote repository location to your local Git repository
    • Type: "git remote add origin repository-url"
